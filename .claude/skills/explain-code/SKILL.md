name: explain-code
description: Explica el código con diagramas visuales y analogías. Úselo para explicar cómo funciona el código, enseñar sobre una base de código o cuando el usuario pregunte "¿cómo funciona esto?".
---

Al explicar código, incluye siempre lo siguiente:

1. **Empieza con una analogía**: Compara el código con algo cotidiano.
2. **Dibuja un diagrama**: Usa arte ASCII para mostrar el flujo, la estructura o las relaciones.
3. **Explica el código paso a paso**: Explica qué sucede.
4. **Resalta un error común**: ¿Cuál es un error o una idea errónea frecuente?

Mantén las explicaciones conversacionales. Para conceptos complejos, usa varias analogías.
